# Phishing Website Detection using ML and NLP

This project detects phishing websites using Machine Learning and Natural Language Processing techniques.

Technologies Used:
- Python
- Scikit-learn
- Pandas
- NLTK
- Logistic Regression
- Naive Bayes

Features:
- URL tokenization using NLP
- Feature extraction
- ML model training
- URL classification as phishing or legitimate

Applications:
- Browser security
- Email spam filtering
- Cybersecurity systems
